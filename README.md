### What confused me
- Difference between region and availability zone
- A cloud Region is a broad, independent geographic area containing multiple data centers, designed for data sovereignty and disaster recovery. An Availability Zone (AZ) is an isolated location within a region, featuring independent power, cooling, and networking to provide high availability and fault tolerance against localized failures.
